# OCR Scanner Web App

## 프로젝트 개요
Firebase Studio로 제작된 바닐라 JavaScript OCR 이미지 스캐너 웹 애플리케이션입니다.

## 주요 기능
- 이미지 업로드 (클릭 또는 드래그앤드롭)
- 이미지 미리보기
- OCR 텍스트 추출 (한글 + 영문 동시 인식)
- OCR 처리 진행률 표시
- 추출된 텍스트 출력 (복사 가능)
- 텍스트 프린트 기능
- 전체 초기화 기능
- 다크/라이트 테마 전환

## 기술 스택
- HTML5
- Vanilla JavaScript (ES6+)
- CSS3 (커스텀 스타일링)
- Tesseract.js v5 (OCR 엔진, CDN 방식)
- Firebase Studio 환경

## 파일 구조
```
/home/user/product-builder/
```

## 개발 히스토리
- 로또 번호 생성기 → 계산기 앱 → OCR 스캐너 앱으로 진화
- 다크/화이트 모드 테마 시스템 유지

## 개발시 주의사항
- 바닐라 JS 유지 (외부 라이브러리 사용 지양)
- 간단하고 명확한 코드 스타일 유지
- Firebase Studio 환경에서 실행됨 (localhost:80)

## 코딩 가이드라인 (Claude 작업 규칙)

### 1. 코딩 스타일
- 모든 변수와 함수 이름은 **카멜 케이스(camelCase)** 사용
- 주석은 **한국어**로 작성 (필요한 경우만)
- 세미콜론(;)을 반드시 사용
- 들여쓰기는 스페이스 4칸

### 2. 기술 스택 제약
- **바닐라 JavaScript만 사용** (React, Vue 등 프레임워크 사용 금지)
- **외부 라이브러리 설치 금지** (새로 설치 필요시 반드시 먼저 물어볼 것)
- CSS는 외부 파일(style.css)에만 작성 (인라인 스타일 금지)

### 3. 개발 규칙
- 새 기능 추가시 기존 코드 스타일을 반드시 따를 것
- 불필요한 리팩토링이나 과도한 추상화 금지
- UI 변경시 **다크/라이트 모드 둘 다** 테스트할 것
- 함수는 가능한 한 짧고 명확하게 작성

### 4. 금지 사항
- ❌ jQuery, Lodash 등 외부 라이브러리 추가
- ❌ 인라인 스타일 (`style="..."` 또는 `element.style.xxx`)
- ❌ 기존 기능을 망가뜨리는 변경
- ❌ 불필요한 주석이나 console.log 남기기

### 5. 변경 전 확인 사항
- 기존 계산 로직이 정상 작동하는지 확인
- 다크/라이트 모드 전환이 정상 작동하는지 확인
- 모든 버튼이 정상 작동하는지 확인
